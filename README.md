# Práctica Final RA1 - Aplicación en Java

Este proyecto consiste en una aplicación de consola que reúne varios pequeños programas en un mismo menú. El usuario podrá seleccionar la función que desea realizar y el programa ejecutará la opción elegida.

Cada una de las funcionalidades está organizada en métodos independientes para mejorar la claridad y facilitar su mantenimiento.

---

## Conversor de Bases Numéricas

Este programa solicita al usuario un número entero en base decimal y muestra su equivalente en:

- **Sistema binario**
- **Sistema hexadecimal**

De esta manera, permite comprender cómo representar un mismo valor en diferentes sistemas de numeración muy utilizados en programación y electrónica.

---

## Simulador de Dados

El usuario indica cuántos dados quiere lanzar.  
Cada dado se genera de manera aleatoria, mostrando un número del 1 al 6.

Esto simula lanzamientos reales de dados y puede servir para juegos, estadísticas o aprendizaje sobre números aleatorios.

---

## Conversor de Temperatura

El programa permite realizar dos tipos de conversión:

- De **Fahrenheit a Celsius**
- De **Celsius a Fahrenheit**

Según la opción elegida, el usuario introduce la temperatura y se realiza el cálculo mostrando el resultado.

---

## Calculador de Edad

El usuario introduce su año de nacimiento y el programa calcula la edad aproximada correspondiente al año actual.

Esta funcionalidad sirve como ejemplo de uso de operaciones aritméticas simples con valores proporcionados por el usuario.

---

## Generador de Saludos

Este programa pide al usuario su nombre y genera un saludo aleatorio entre varias opciones posibles.

Cada ejecución puede mostrar un mensaje distinto, lo que lo hace más interactivo y dinámico.
